# Python Crash Course

## 📌 Objective
Learn the core fundamentals of Python, including printing output, variables, data types, Boolean values, arithmetic operations, comparison operators, comments, and whitespace characters.

---

## 📖 Topics Covered

- Python Character Set
- `print()` Function
- Letters, Digits, and Special Symbols
- Whitespaces (`\t`, `\n`)
- Variables
- Data Types (`str`, `int`, `float`, `bool`, `NoneType`)
- `None` Value
- Arithmetic Operators (`+`, `-`, `*`, `/`, `%`, `**`, `//`)
- Comparison (Relational) Operators (`==`, `!=`, `>`, `<`, `>=`, `<=`)
- Boolean Expressions
- `type()` Function
- Single-line Comments (`#`)
- Multi-line Comments (`''' '''` or `""" """`)

---

## 💻 Key Examples

### Print Output

```python
print("Hello World!")
```

### Variables

```python
name = "Ali"
age = 23
price = 78.54
is_student = True
phone_number = None
```

### Data Types

```python
print(type(name))
print(type(age))
print(type(price))
print(type(is_student))
print(type(phone_number))
```

### Boolean Expression

```python
result = (2 + 3) == (3 + 2)
print(result)   # True
```

### Arithmetic Operators

```python
a = 30
b = 50

print(a + b)    # Addition
print(a - b)    # Subtraction
print(a * b)    # Multiplication
print(a / b)    # Division
print(a % b)    # Modulus
print(a ** 2)   # Exponent
print(a // b)   # Floor Division
```

### Comparison Operators

```python
a = 2
b = 4

print(a == b)
print(a != b)
print(a > b)
print(a < b)
print(a >= b)
print(a <= b)
```

### Whitespace Characters

```python
print("Hello\tWorld")
print("Hello\nWorld")
```

### Comments

```python
# This is a single-line comment

"""
This is a
multi-line comment.
"""
```

---

## 🎯 Learning Outcomes

After completing these topics, students will be able to:

- Display output using the `print()` function.
- Understand Python character sets.
- Use letters, digits, special symbols, and whitespace characters.
- Create and use variables.
- Identify Python data types.
- Use the `None` value.
- Perform arithmetic operations.
- Compare values using comparison operators.
- Evaluate Boolean expressions.
- Check the data type of a variable using `type()`.
- Write single-line and multi-line comments.
